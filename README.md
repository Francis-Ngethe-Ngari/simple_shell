# **SIMPLE_SHELL**

## **Overview of shell**

Shell provides you with an interface to the Unix System. Gathers input from user and excutes programs based on input.

When a program finishes executing it displays program output.

### **What is shell ?**

Shell is a command-line interface that allows users to interact with an operating system.

It serves as an intermediary between the user and the operating system, providing a way to execute commands and manage

files and processes.
